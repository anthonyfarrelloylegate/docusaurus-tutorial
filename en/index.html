What is Watson Care Manager Reporting?

With WCM Reporting, Watson Care Manager users can run reports, access dashboards, and create custom reports and dashboards based on data that is recorded in the Care Team application. The IBM Watson Care Manager Reporting application is built on IBM Cognos Analytics
